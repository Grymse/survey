# Impulse purchase survey

- Allowed me to create a survey with specific features, that would otherwise cost me a fortune...

# Technologies

- ReactJS
- Firebase
- Shadcn UI + Tailwind

# HOW TO START

- Ensure you have node 16+
- Download repo
- In impulse-survey folder run
  - `npm install`
  - `npm run dev`
- Open the link `http://localhost:????/`
